# 🚀 Task Creation App

> A simple **Task Management Web Application** built using **HTML, CSS,
> and JavaScript**. It allows users to create, edit, and delete tasks
> through a clean and interactive interface.

------------------------------------------------------------------------

# ✨ Features

✅ Create new tasks\
✏️ Edit existing tasks\
🗑️ Delete tasks\
🎯 Assign priority (Low, Medium, High, Urgent)\
📋 Display tasks as cards\
🎨 Modern dark-themed UI

------------------------------------------------------------------------

# 🛠️ Technologies Used

-   🌐 HTML5
-   🎨 CSS3
-   ⚡ JavaScript (Vanilla JS)

------------------------------------------------------------------------

# 📂 Project Structure

``` text
📁 Project
│── index.html
│── style.css
└── script.js
```

### 🌐 index.html

Responsible for the structure of the application.

It contains: - 🧭 Navigation bar - ➕ Create Task button - 📝 Task
form - 📦 Container where task cards appear

------------------------------------------------------------------------

### 🎨 style.css

Responsible for making the application attractive.

It provides: - 🌙 Dark theme - 📱 Responsive Flexbox layout - 🎴
Beautiful task cards - 🎯 Button colors for different actions

------------------------------------------------------------------------

### ⚡ script.js

This is the **brain of the application**.

It controls everything the user does: - Reading form inputs - Creating
task objects - Saving tasks in an array - Showing task cards - Editing
tasks - Deleting tasks

------------------------------------------------------------------------

# 🔄 How the App Works

``` text
👤 User Clicks "Create Task"
           │
           ▼
📝 Form Opens
           │
           ▼
✍️ User Enters Details
           │
           ▼
✅ Validation Checks
           │
           ▼
📦 Task Stored in Array
           │
           ▼
🎴 Task Card Displayed
           │
           ▼
✏️ Edit or 🗑️ Delete Anytime
```
